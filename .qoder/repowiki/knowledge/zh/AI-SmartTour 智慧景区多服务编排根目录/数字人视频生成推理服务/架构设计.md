单文件服务（server.py）+ Docker 容器化部署，整体为请求-任务-下载三段式异步架构：
- HTTP 层：FastAPI 暴露 /health、/generate、/jobs/{id}、/jobs/{id}/video 四个端点，使用 lifespan 钩子完成启动日志与过期任务清理。
- 任务调度：POST /generate 将图片与文本参数持久化为 job 记录（内存 dict），并通过 asyncio.create_task 提交后台 _generate_video 协程处理，实现非阻塞响应。
- 引擎路由：_generate_video 根据进程级 HAS_GPU 标志选择 _musetalk_generate 或 _ffmpeg_simulate；当前 MuseTalk 推理尚未接入，GPU 环境也会降级到 ffmpeg 模拟路径。
- 存储边界：输出目录由 OUTPUT_DIR 环境变量控制（默认 /data/outputs），每个 job 独立子目录存放 reference.png 与 output.mp4，通过 FileResponse 直接返回磁盘文件。
- 依赖方向：仅依赖 fastapi + uvicorn + python-multipart，外部工具 ffmpeg 通过 asyncio.create_subprocess_exec 调用，不引入 Python 多媒体库。
