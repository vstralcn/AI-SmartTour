模块按三层划分：
- `api/chat.py`：FastAPI 路由层，暴露 `/agent/capabilities`、`/sessions`（REST）和 `/chat/stream`（WebSocket）三个端点，负责解析请求、调用 `dialogue_engine`、推送流式事件并持久化交互记录。
- `core/dialogue.py`：对话引擎 `DialogueEngine`，维护每会话的消息历史（最近10条），编排 LLM 调用；当 `settings.llm_api_key` 为空时回退到 RAG 直接答案，否则通过 `openai.AsyncOpenAI` 以 `stream=True` 分片输出。
- `core/agent.py`：轻量导览 Agent `GuideAgent`，内置白名单工具集（knowledge / route / profile / feedback），通过关键词规则完成意图分类、游客画像更新、时长提取与路线规划，不依赖外部模型做决策。

依赖方向单向：`api → dialogue → agent → rag/recommend/sentiment/persistence`，LLM 客户端在 `_call_llm_stream` 中按需 import，避免无配置时加载失败。全局单例 `dialogue_engine = DialogueEngine()` 作为模块级入口被 API 直接引用。
